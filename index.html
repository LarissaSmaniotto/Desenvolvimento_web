Exercício 1 (optatório)

Crie uma nova página para cada lista. Deve conter um link para cada uma na página index.html

Faça 4 listas contendo 5 itens em cada uma.

A primeira lista deve ser não ordenada, utilizando bolinhas em cada item

A segunda lista deve ser ordenada porém com número romanos

A terceira lista deve ser não ordenada com quadrado indicando cada item e deve conter uma sublista com circulos vazios (não é bolinha)

A quarta lista deve ser não ordenada porém com uma sublista ordenada em caracteres alpha (A. B. C. D) em maiúsculo

Valor: 1 ponto a ser adicionado na avaliação 1.

Referência: https://www.w3schools.com/html/html_lists.asp
